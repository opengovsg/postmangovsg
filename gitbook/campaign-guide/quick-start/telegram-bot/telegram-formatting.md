# Telegram Formatting

**Bold**

```
<b>PostmanSG</b>
```

Underline

```
<u>PostmanSG</u>
```

_Italic_

```
<i>PostmanSG</i>
```

Hyperlink ([PostmanSG](https://postman.gov.sg/))

```
<a href="https://postman.gov.sg">postman</a>
```
